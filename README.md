# Análise em Excel - Superstore Dataset (Kaggle e Tableu)
Análise utilizando o excel dos dados públicos para fins educacionais "Superstore Dataset" inicialmente disponível pela Tableu e posteriormente divulgado em outras plataformas como Kaggle. Esse projeto é apenas para fins educacionais e os dados não são utilizados para fins comerciais.

![Parte 1 ](images/dashboard.png)
 *  Apenas visualização dos dados através de uma tabela dinâmica (pivot table). Foi usado o WPS 2019 para ler o dataset e fazer a visualização.
 *  Dados genéricos anonimizados de uma superloja locada nos Estados Unidos com faturamentos registrados em várias cidades;
 *  Existem várias informações interessantes presentes nos dados: o faturamento bruto, o lucro, IDs de clientes únicas que permitem identificar cada cliente, estados, datas de envio, tipo de categoria de envio (se envio rápido ou comum), segmentos se é B2B ou B2C, etc. A partir desses dados já é possível imaginar várias análises sobre as vendas dessa super loja.
